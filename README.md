# wix-exotics-3-fronti-res
Site web front avec le backend sur Wix pour Exotics 3 Frontières.

## Fonctionnalités
- Intégration Wix Headless (Produits, Panier, Checkout)
- Design moderne avec Tailwind CSS
- Animations fluides avec Framer Motion
- Dashboard Admin pour la configuration du site
- Entièrement responsive et optimisé pour le SEO

## Débuter

Tout d'abord, installez les dépendances :

```bash
npm install
```

Ensuite, lancez le serveur de développement :

```bash
npm run dev
```

Ouvrez [http://localhost:3000](http://localhost:3000) dans votre navigateur pour voir le résultat.

## Déploiement

Le site est optimisé pour être déployé sur [Vercel](https://vercel.com). Assurez-vous de configurer la variable d'environnement `NEXT_PUBLIC_WIX_CLIENT_ID`.
